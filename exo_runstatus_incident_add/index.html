exo runstatus incident add

Add an incident to a runstat.us page
exo runstatus incident add [page name] [flags]

Options

--description, -d - Description for the initial creation event
--help, -h - help for add
--services - List of strings with the services impacted. e.g: <service1,service2,…>
--state, -s - <major_outage | partial_outage | degraded_performance | operational
--status - <investigating | identified | monitoring>
--title, -t - Title of the incident

Options inherited from parent commands

--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
--output-format, -O - Output format, can be table|json|text
--output-template - Template to use if output format is "template"
--use-account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
