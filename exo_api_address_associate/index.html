exo api address associate

Acquires and associates a public IP to an account.

Synopsis

Acquires and associates a public IP to an account. http://cloudstack.apache.org/api/apidocs-4.4/user/associateIpAddress.html

exo api address associate [flags]

Options

--interval - healthcheck definition: time in seconds to wait for each check. Default: 10, minimum: 5
--mode - healthcheck definition: healthcheck mode can be either 'tcp' or 'http'
--path - healthcheck definition: the path against which the 'http' healthcheck will be performed. Required if mode is 'http', ignored otherwise.
--port - healthcheck definition: the port against which the healthcheck will be performed. Required if a 'mode' is provided.
--strikes-fail - healthcheck definition: number of times to retry before declaring the healthcheck 'dead'. Default: 3
--strikes-ok - healthcheck definition: number of times to retry before declaring the healthcheck 'alive'. Default: 2
--timeout - healthcheck definition: time in seconds to wait for each check. Default: 2, cannot be greater than interval.
--zoneid - the ID of the availability zone you want to acquire a public IP address from
--help, -h - help for associate
