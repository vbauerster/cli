exo eip update

update EIP
exo eip update [eip ID] [flags]

Options

--healthcheck-interval - the time in seconds to wait for between each healthcheck.
--healthcheck-mode - the healthcheck type. Should be tcp or http.
--healthcheck-path - the healthcheck path. Required if mode is http.
--healthcheck-port - the healthcheck port (e.g. 80 for http).
--healthcheck-strikes-fail - the number of times to retry before declaring the IP dead.
--healthcheck-strikes-ok - the number of times to retry before declaring the IP healthy.
--healthcheck-timeout - the timeout in seconds to wait for each check (default is 2). Should be lower than the interval.
--help, -h - help for update

Options inherited from parent commands

--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
--output-format, -O - Output format, can be table|json|text
--output-template - Template to use if output format is "template"
--use-account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
