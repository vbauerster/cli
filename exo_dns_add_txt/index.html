exo dns add TXT

Add TXT record type to a domain

Synopsis

Add a «TXT» record. This is useful for domain records that are not covered by
the standard record types. For example, Google uses this type of record for domain verification.

exo dns add TXT <domain name> [flags]

Options

--content, -c - Content record
--help, -h - help for TXT
--name, -n - Leave this blank to create a record for <domain name>, You may use the '*' wildcard here.
--ttl, -t - The time in seconds to live (refresh rate) of the record.

Options inherited from parent commands

--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
--output-format, -O - Output format, can be table|json|text
--output-template - Template to use if output format is "template"
--use-account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
