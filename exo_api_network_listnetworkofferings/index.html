exo api network listNetworkOfferings

Lists all available network offerings.

Synopsis

Lists all available network offerings. http://cloudstack.apache.org/api/apidocs-4.4/user/listNetworkOfferings.html

exo api network listNetworkOfferings [flags]

Options

--availability - the availability of network offering. Default value is Required
--displaytext - list network offerings by display text
--guestiptype - list network offerings by guest type: Shared or Isolated
--id - list network offerings by id
--isdefault - true if need to list only default network offerings. Default value is false
--istagged - true if offering has tags specified
--keyword - List by keyword
--name - list network offerings by name
--networkid - the ID of the network. Pass this in if you want to see the available network offering that a network can be changed to.
--page -
--pagesize -
--sourcenatsupported - true if need to list only netwok offerings where source nat is supported, false otherwise
--specifyipranges - true if need to list only network offerings which support specifying ip ranges
--specifyvlan - the tags for the network offering.
--state - list network offerings by state
--tags - list network offerings by tags
--traffictype - list by traffic type
--zoneid - list network offerings available for network creation in specific zone
--help, -h - help for listNetworkOfferings
