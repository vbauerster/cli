exo vm create

Create and deploy a virtual machine
exo vm create <vm name>+ [flags]

Options

--anti-affinity-group, -a - <name | id, name | id, …>
--cloud-init-file, -f - Deploy instance with a cloud-init file
--disk, -d - <disk size>
--help, -h - help for create
--ipv6, -6 - enable ipv6
--keypair, -k - <ssh keys name>
--privnet, -p - <name | id, name | id, …>
--security-group, -s - <name | id, name | id, …>
--service-offering, -o - <name | id> (micro|tiny|small|medium|large|extra-large|huge|mega|titan|jumbo)
--template, -t - <template name | id> (default: Linux Ubuntu 18.04 LTS 64-bit)
--template-filter - The template filter to use (mine,community,featured)
--zone, -z - <zone name | id> (ch-dk-2|ch-gva-2|at-vie-1|de-fra-1|bg-sof-1)

Options inherited from parent commands

--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
--output-format, -O - Output format, can be table|json|text
--output-template - Template to use if output format is "template"
--use-account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
