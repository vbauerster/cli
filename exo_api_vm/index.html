exo api vm

Virtual machine management
exo api vm [command] [flags]

Options

--help, -h - help for vm

Options inherited from parent commands

--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
--debug, -d - debug mode on
--dry-run, -D - produce a cURL ready URL
--output-format, -O - Output format, can be table|json|text
--output-template - Template to use if output format is "template"
--use-account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]

SEE ALSO

  • exo api - Exoscale api

  • exo api vm addNic - Adds VM to specified network by creating a NIC

  • exo api vm attachIso - Attaches an ISO to a virtual machine.

  • exo api vm changeService - Changes the service offering for a virtual machine. The virtual machine must be in a «Stopped» state for this command to take effect.

  • exo api vm deleteReverseDNSFromVM - Delete the PTR DNS record(s) from the virtual machine

  • exo api vm deploy - Creates and automatically starts a virtual machine based on a service offering, disk offering, and template.

  • exo api vm destroy - Destroys a virtual machine.

  • exo api vm detachIso - Detaches any ISO file (if any) currently attached to a virtual machine.

  • exo api vm expunge - Expunge a virtual machine. Once expunged, it cannot be recoverd.

  • exo api vm getPassword - Returns an encrypted password for the VM

  • exo api vm getUserData - Returns user data associated with the VM

  • exo api vm list - List the virtual machines owned by the account.

  • exo api vm queryReverseDNSForVM - Query the PTR DNS record(s) for the virtual machine

  • exo api vm reboot - Reboots a virtual machine.

  • exo api vm recover - Recovers a virtual machine.

  • exo api vm removeNic - Removes VM from specified network by deleting a NIC

  • exo api vm resetPassword - Resets the password for virtual machine. The virtual machine must be in a «Stopped» state and the template must already support this feature for this command to take effect.

  • exo api vm restore - Restore a VM to original template/ISO or new template/ISO

  • exo api vm scale - Scales the virtual machine to a new service offering.

  • exo api vm start - Starts a virtual machine.

  • exo api vm stop - Stops a virtual machine.

  • exo api vm update - Updates properties of a virtual machine. The VM has to be stopped and restarted for the new properties to take effect. UpdateVirtualMachine does not first check whether the VM is stopped. Therefore, stop the VM manually before issuing this call.

  • exo api vm updateReverseDNSForVM - Update/create the PTR DNS record(s) for the virtual machine

  • exo api vm updateVMAffinityGroup - Updates the affinity/anti-affinity group associations of a virtual machine. The VM has to be stopped and restarted for the new properties to take effect.

  • exo api vm updateVMNicIP - Update the default Ip of a VM Nic
