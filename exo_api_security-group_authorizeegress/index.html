exo api security-group authorizeEgress

Authorize a particular ingress/egress rule for this security group

Synopsis

Authorize a particular ingress/egress rule for this security group http://cloudstack.apache.org/api/apidocs-4.4/user/authorizeSecurityGroupEgress.html

exo api security-group authorizeEgress [flags]

Options

--cidrlist - the cidr list associated
--description - the description of the ingress/egress rule
--endport - end port for this ingress/egress rule
--icmpcode - error code for this icmp message
--icmptype - type of the icmp message being sent
--protocol - TCP is default. UDP, ICMP, ICMPv6, AH, ESP, GRE, IPIP are the other supported protocols
--securitygroupid - The ID of the security group. Mutually exclusive with securitygroupname parameter
--securitygroupname - The name of the security group. Mutually exclusive with securitygroupid parameter
--startport - start port for this ingress/egress rule
--usersecuritygrouplist - user to security group mapping
--help, -h - help for authorizeEgress
